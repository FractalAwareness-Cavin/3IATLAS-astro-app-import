3I/ATLAS ΚΑΙ ΕΦΑΡΜΟΓΕΣ ΧΩΡΙΣ ΕΙΣΑΓΩΓΗ
====================================

Το παρόν σημείωμα περιγράφει τους τρέχοντες περιορισμούς των **Astro Gold** (macOS, iOS/iPadOS)
και **TimePassages Desktop** (macOS / Windows), καθώς και τρόπους χρήσης των ενσωματωμένων
λειτουργιών «extra point».

## Astro Gold (macOS / iOS / iPadOS)
- Το Astro Gold επιτρέπει μόνο την ενεργοποίηση/απενεργοποίηση του καταλόγου σωμάτων που προσφέρει ο κατασκευαστής. Δεν υπάρχει τεκμηριωμένη δυνατότητα εισαγωγής εξωτερικών επιμηρίδων ή στοιχείων τροχιάς.
- Μπορείτε να ενεργοποιήσετε τα πρόσθετα του προμηθευτή μέσω:
  - **macOS**: `Astro Gold → Preferences → Displayed Points → Add Extra Points…`
  - **iOS / iPadOS**: `Settings → Chart Points → Add Extra Points…`
- Τα «Custom Points» του Astro Gold είναι σταθερές εκλειπτικές μακρότητες που εισάγετε χειροκίνητα. Χρήσιμες για αστέρες αναφοράς ή συγκεκριμένους βαθμούς, όμως δεν αλλάζουν με τον χρόνο.
- Προχωρημένοι χρήστες μπορούν να δουν τους φακέλους υποστήριξης (π.χ. `~/Library/Application Support/com.ajnaware.Astro-Gold` σε macOS ή `~/Documents/Astro Gold` για χάρτες), αλλά δεν δέχονται **κινούμενες** επιμηρίδες για «σύρσιμο και απόθεση».

**Συμπέρασμα:** Μέχρι η Esoteric Technologies να προσθέσει το 3I/ATLAS στον κατάλογό της (ή να διαθέσει μηχανισμό εισαγωγής), δεν γίνεται να το εγκαταστήσετε ως κινούμενο σώμα στο Astro Gold. Χρησιμοποιήστε τα ενσωματωμένα extras ή καταχωρίστε το 3I/ATLAS ως σταθερό custom point αν χρειάζεστε τη θέση του σε συγκεκριμένη εποχή.

### Οδηγός σταθερού σημείου (Astro Gold)
1. Επιλέξτε ημερομηνία/ώρα (π.χ. **2025-10-29 00:00 UT** κοντά στο περιήλιο).
2. Ανοίξτε `apps-using-csv-files/geocentric_daily.csv`, εντοπίστε τη γραμμή και σημειώστε τροπική μακρότητα (`lambda_deg`) και πλάτος (`beta_deg`). Για το παράδειγμα: `lambda_deg = 203.560 deg` (≈ 23°33' Σκορπιού) και `beta_deg = 2.283 deg`.
3. Στο Astro Gold (macOS) μεταβείτε **Astro Gold → Preferences → Displayed Points → Add Extra Points…** και στην καρτέλα **Custom Points**. Σε iOS/iPadOS χρησιμοποιήστε **Settings → Chart Points → Add Extra Points…**.
4. Δημιουργήστε νέο σημείο (π.χ. `3I/ATLAS 2025-10-29 UT`), εισάγετε τη μακρότητα σε δεκαδικό ή στο απαιτούμενο format και (προαιρετικά) καταγράψτε πλάτος/απόσταση.
5. Αποθηκεύστε. Το σημείο μένει σταθερό· επαναλάβετε με νέα μακρότητα όταν χρειάζεται άλλη εποχή.

## TimePassages Desktop (macOS / Windows)
- Το TimePassages επιτρέπει ενεργοποίηση σωμάτων που συνοδεύουν την εφαρμογή (μεγάλοι αστεροειδείς, κένταυροι, Eris/TNO κ.λπ.). Ροή εργασίας:
  - `Preferences → Edit Chart Points` (macOS) ή `Edit → Chart Points…` (Windows) για ενεργοποίηση κατηγοριών.
  - `Display → Chart Points…` για να επιβεβαιώσετε ότι εμφανίζονται στους χάρτες.
- Η λειτουργία **Custom Points** είναι επίσης σταθερή (π.χ. το Γαλαξιακό Κέντρο στα 27° Τοξότη). Προς το παρόν δεν υπάρχει μηχανισμός εισαγωγής νέων κινούμενων αντικειμένων.
- Οι φάκελοι δεδομένων που είναι ορατοί (π.χ. `~/Library/Application Support/TimePassages/` σε macOS ή `%APPDATA%\TimePassages\` στα Windows) αποθηκεύουν μόνο ρυθμίσεις και χάρτες· μια επιμηρίδα δεν μπορεί να προστεθεί εκεί.

**Συμπέρασμα:** Το TimePassages δεν μπορεί ακόμη να παρακολουθήσει αυτόματα το 3I/ATLAS. Αν μελλοντική έκδοση προσθέσει import, οι οδηγίες θα ενημερωθούν εδώ. Προς το παρόν μπορείτε να δημιουργήσετε σταθερό custom point για συγκεκριμένο Julian Date.

### Οδηγός σταθερού σημείου (TimePassages)
1. Ορίστε την επιθυμητή εποχή (π.χ. **2025-10-18 00:00 UT** κοντά στη μέγιστη επιμήκυνση).
2. Δείτε `apps-using-csv-files/geocentric_daily.csv`. Η γραμμή της 2025-10-18 δείχνει `lambda_deg = 209.285 deg` (≈ 29°17' Ζυγού) και `beta_deg = 2.715 deg`.
3. Στο TimePassages ανοίξτε **Edit → Chart Points…** (Windows) ή **Preferences → Edit Chart Points** (macOS), μεταβείτε στα **Custom Points** και πατήστε **Add**.
4. Δώστε όνομα (π.χ. `3I/ATLAS 2025-10-18 UT`) και εισάγετε τη μακρότητα στο format της εφαρμογής. Προσθέστε πλάτος/απόσταση στην περιγραφή αν θέλετε.
5. Αποθηκεύστε και ενεργοποιήστε το σημείο. Ενημερώστε το με νέα μακρότητα όταν χρειαστείτε άλλο στιγμιότυπο.
