3I/ATLAS — Γρήγορη εισαγωγή στο WinStars
========================================

Το WinStars (v3) μπορεί να λάβει στοιχεία κομητών MPC μέσω του ενσωματωμένου editor.

### Μέθοδος 1 — επικόλληση της γραμμής
1. Εκκινήστε το WinStars.
2. Μεταβείτε **Preferences → Solar system → Import orbital elements** (ή **Add object**).
3. Επιλέξτε **MPC single line** και επικολλήστε το περιεχόμενο του `3I_ATLAS_mpc_1line.txt`.
4. Επιβεβαιώστε και βεβαιωθείτε ότι το 3I/ATLAS είναι ενεργό στη λίστα σωμάτων.

### Μέθοδος 2 — αντικατάσταση αρχείου κομητών (για προχωρημένους)
1. Κλείστε το WinStars.
2. Κρατήστε αντίγραφο του αρχείου κομητών:
   - Windows: `%APPDATA%\WinStars3\databases\comets.txt`
   - Linux: `~/.config/WinStars3/databases/comets.txt`
3. Προσθέστε τη γραμμή από το `3I_ATLAS_mpc_1line.txt` σε αυτό το αρχείο.
4. Ξεκινήστε ξανά το WinStars· το 3I/ATLAS θα βρίσκεται στη λίστα κομητών.

Το αρχείο μίας γραμμής βασίζεται στη λύση 27 του JPL SBDB (2025-10-10). Ενημερώστε το όταν υπάρξει νέα λύση.
