ΕΡΓΑΛΕΙΟΘΗΚΗ ΠΡΟΓΡΑΜΜΑΤΙΣΤΗ
===========================

Αυτός ο φάκελος περιλαμβάνει όλα όσα χρειάζονται για να αναδημιουργήσετε το
πακέτο επιμηρίδων από την αρχή.

Περιεχόμενα
-----------
- `raw/` – αποκρίσεις JSON που ελήφθησαν από το NASA/JPL Horizons API.
- `scripts/generate_ephemeris.py` – script Python που ανακατασκευάζει τα CSV,
  τα αρχεία κειμένου του Solar Fire, την έξοδο MPC και τις περιλήψεις εισόδου στα ζώδια.
- `scripts/build_swisseph.sh` – βοηθητικό εργαλείο που μετατρέπει τους πίνακες CSV σε
  δυαδικά Swiss Ephemeris `.se1` (απαιτεί `mksweph`).
- `vendor/` – ενσωματωμένο module `pyswisseph`, ώστε τα αστρικά αποτελέσματα να λειτουργούν
  χωρίς επιπλέον εγκαταστάσεις.

Αναδημιουργία
-------------
```
cd developer/scripts
python3 generate_ephemeris.py
```
Το script κάνει νέο ερώτημα στο Horizons (απαιτείται σύνδεση) και ενημερώνει
τους φακέλους στη ρίζα του αποθετηρίου.

Δυαδικά Swiss Ephemeris
-----------------------
Αν διαθέτετε το ιδιόκτητο εργαλείο `mksweph`:
```
cd developer/scripts
bash build_swisseph.sh
```
Τα αρχεία `.se1` γράφονται στο `../swisseph/` προς διανομή.

Μπορείτε να τροποποιήσετε το script για να αλλάξετε συχνότητα, συστήματα συντεταγμένων
ή κεντρικά σώματα· τα ακατέργαστα JSON παρέχουν ιχνηλασιμότητα μέχρι το Horizons.

Για αναλυτικές οδηγίες, συμπεριλαμβανομένου του ενημερωτή SBDB και των βοηθητικών ελέγχων,
δείτε το `scripts/README.md`.
