3I/ATLAS И ПРИЛОЖЕНИЯ БЕЗ ИМПОРТА
=================================

Здесь описаны ограничения **Astro Gold** (macOS, iOS/iPadOS) и **TimePassages Desktop** (macOS / Windows), а также способы использовать встроенные функции "extra point".

## Astro Gold (macOS / iOS / iPadOS)
- Astro Gold позволяет только включать или выключать каталог тел, поставляемых разработчиком. Документированной возможности импортировать сторонние эфемериды или орбитальные элементы нет.
- Включить встроенные дополнительные точки можно через:
  - **macOS**: `Astro Gold → Preferences → Displayed Points → Add Extra Points…`
  - **iOS / iPadOS**: `Settings → Chart Points → Add Extra Points…`
- «Custom Points» в Astro Gold — это фиксированные эклиптические долготы, которые вводятся вручную. Они подходят для опорных звезд или конкретных градусных отметок, но не изменяются со временем.
- Продвинутые пользователи могут просмотреть служебные папки (`~/Library/Application Support/com.ajnaware.Astro-Gold` на macOS или `~/Documents/Astro Gold` для карт), но туда **нельзя** просто положить эфемериду движущегося тела.

**Итог:** Пока Esoteric Technologies не добавит 3I/ATLAS в каталог (или не откроет импорт), установить его как движущийся объект в Astro Gold нельзя. Используйте встроенные дополнительные точки или создайте фиксированный custom point, если нужна позиция на конкретную дату.

### Пошагово: фиксированная точка (Astro Gold)
1. Выберите дату/время (например, **2025-10-29 00:00 UT**, около перигелия).
2. Откройте `apps-using-csv-files/geocentric_daily.csv`, найдите нужную строку и перепишите тропическую долготу (`lambda_deg`) и широту (`beta_deg`). В примере: `lambda_deg = 203.560 deg` (≈ 23°33' Скорпиона) и `beta_deg = 2.283 deg`.
3. В Astro Gold (macOS) перейдите к **Astro Gold → Preferences → Displayed Points → Add Extra Points…**, затем вкладка **Custom Points**. На iOS/iPadOS: **Settings → Chart Points → Add Extra Points…**.
4. Создайте точку, например `3I/ATLAS 2025-10-29 UT`, введите долготу в десятичном виде (или в формате знак/градусы) и при необходимости добавьте широту/дистанцию в заметки.
5. Сохраните точку. Она статична; повторяйте шаги со свежей долготой для новых эпох.

## TimePassages Desktop (macOS / Windows)
- TimePassages позволяет включать встроенные тела (крупные астероиды, кентавры, Eris/TNO и т. п.). Процедура:
  - `Preferences → Edit Chart Points` (macOS) или `Edit → Chart Points…` (Windows) для активации категорий.
  - `Display → Chart Points…` чтобы убедиться, что они отображаются.
- Функция **Custom Points** также фиксирует градус (например, можно добавить Галактический центр на 27° Стрельца). На данный момент TimePassages не поддерживает импорт новых движущихся объектов.
- Пользовательские каталоги (`~/Library/Application Support/TimePassages/` на macOS или `%APPDATA%\TimePassages\` на Windows) содержат только настройки и сохраненные карты; туда нельзя подложить эфемериду для автоподхвата.

**Итог:** TimePassages пока не может отслеживать 3I/ATLAS автоматически. Если в будущем появится импорт, инструкции будут обновлены. Пока используйте фиксированный custom point на выбранную дату.

### Пошагово: фиксированная точка (TimePassages)
1. Выберите эпоху (например **2025-10-18 00:00 UT**, около максимального удлинения).
2. Посмотрите координаты в `apps-using-csv-files/geocentric_daily.csv`. Для 2025-10-18 указано `lambda_deg = 209.285 deg` (≈ 29°17' Весов) и `beta_deg = 2.715 deg`.
3. В TimePassages откройте **Edit → Chart Points…** (Windows) или **Preferences → Edit Chart Points** (macOS), перейдите к **Custom Points** и нажмите **Add**.
4. Введите название, например `3I/ATLAS 2025-10-18 UT`, и долготу в формате знак/градусы, принятом в программе. Широту или расстояние можно занести в описание.
5. Сохраните и активируйте точку. Обновляйте долготу при необходимости нового снимка.
