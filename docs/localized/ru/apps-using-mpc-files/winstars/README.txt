3I/ATLAS — быстрый импорт в WinStars
====================================

WinStars (v3) может загружать кометные элементы MPC через встроенный редактор.

### Метод 1 — вставить однострочный элемент
1. Запустите WinStars.
2. Откройте **Preferences → Solar system → Import orbital elements** (или **Add object**).
3. Выберите **MPC single line** и вставьте содержимое `3I_ATLAS_mpc_1line.txt`.
4. Подтвердите и убедитесь, что 3I/ATLAS включён в списке отображаемых объектов.

### Метод 2 — заменить файл комет (для опытных пользователей)
1. Закройте WinStars.
2. Создайте резервную копию файла комет:
   - Windows: `%APPDATA%\WinStars3\databases\comets.txt`
   - Linux: `~/.config/WinStars3/databases/comets.txt`
3. Добавьте однострочную запись из `3I_ATLAS_mpc_1line.txt` в этот файл.
4. Перезапустите WinStars — 3I/ATLAS появится в списке комет.

Файл с однострочными данными основан на решении JPL SBDB №27 (2025-10-10). Обновляйте его при выходе новой орбитальной модели.
