НАБОР ИНСТРУМЕНТОВ ДЛЯ РАЗРАБОТЧИКОВ
===================================

Этот каталог содержит всё необходимое, чтобы заново собрать пакет
эфемерид с нуля.

Содержимое
----------
- `raw/` – JSON-ответы, полученные через API NASA/JPL Horizons.
- `scripts/generate_ephemeris.py` – Python-скрипт, который пересоздаёт CSV,
  текстовые файлы Solar Fire, вывод MPC и сводки входов в знаки.
- `scripts/build_swisseph.sh` – помощник для преобразования CSV-таблиц в бинарные
  файлы Swiss Ephemeris `.se1` (требуется `mksweph`).
- `vendor/` – включённый модуль `pyswisseph`, чтобы сидерические данные работали
  без дополнительных установок.

Регенерация
-----------
```
cd developer/scripts
python3 generate_ephemeris.py
```
Скрипт повторно обращается к Horizons (нужен интернет) и обновляет
каталоги в корне репозитория.

Бинарники Swiss Ephemeris
-------------------------
Если у вас есть доступ к проприетарной утилите `mksweph`:
```
cd developer/scripts
bash build_swisseph.sh
```
`.se1`-файлы будут записаны в `../swisseph/` для распространения.

При необходимости адаптируйте скрипт: меняйте шаг по времени, системы координат
или центральные тела; исходные JSON-файлы обеспечивают отслеживаемость до Horizons.

Подробные инструкции, включая обновление SBDB и проверки, приведены в
`scripts/README.md`.
