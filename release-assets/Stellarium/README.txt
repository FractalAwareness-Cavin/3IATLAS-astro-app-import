3I/ATLAS — Stellarium Quick Import
==================================

Stellarium imports comet elements from MPC one-line files.

Files
-----
- `3I_ATLAS_mpc_1line.txt` — MPC comet entry for C/2025 N1 (ATLAS).

Steps
-----
1. Enable the Solar System Editor plugin (**Configuration → Plugins → Solar System Editor**, tick **Load at startup**, restart if needed).
2. Open **Solar System Editor → Configure → Solar System → Import elements in MPC format → File**.
3. Select `3I_ATLAS_mpc_1line.txt`, click **Open**, then **Add object(s)**.
4. Use the search box to find `3I/ATLAS`.
