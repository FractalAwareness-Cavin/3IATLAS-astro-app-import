3I/ATLAS — Stellarium Quick Import
==================================

Stellarium accepts comet elements in MPC one-line format. This bundle contains the line generated from **JPL SBDB solution 27 (2025-10-10)**.

Files
-----
- `3I_ATLAS_mpc_1line.txt` — MPC comet entry for C/2025 N1 (ATLAS).

Steps
-----
1. Press `F2` (or open **Configuration**), go to **Plugins → Solar System Editor**, tick **Load at startup**, and click **Configure**. Restart Stellarium if you just enabled the plugin.
2. After the restart, revisit **Solar System Editor → Configure → Solar System** and choose **Import elements in MPC format → File**.
3. Select `3I_ATLAS_mpc_1line.txt`, keep **Object type** set to *Comet*, confirm the name `3I/ATLAS`, and click **Add object(s)**.
4. Close the dialogs, press `F3`, search for `3I/ATLAS`, and mark it as a favourite if desired. If the search fails, restart Stellarium so it reloads the catalogue.
